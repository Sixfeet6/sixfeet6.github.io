用函数实现张量交换指标

注意：

如果用指针形式传递数组，函数内部也必须把数组当作指针对待，也就是说，必须手工计算偏移量

and,

书写交换张量的偏移量时，指标和对应的维的长度都要交换

知识点：

运算符

一元运算符：

symbol meaning
++ add itself
-- minus itself
+ positive
- negative
! wrong
~ opposite
& get address
* quote varible
sizeof() return the bytes number
(type) change type

Binary Operators:

算术运算符：

symbol meaning
+ plus
- minus
* multiply
% mod
/

关系运算符：

symbol meaning
== equal
!= not equal
> greater
< less

逻辑运算符：

symbol meaning
&& and
|| or

位运算符：

symbol meaning
& bit and
| bit or
^ 按位异或
<< move left
>> move right

赋值运算符：略

三元运算符：

only this one:

x()y? x:y

文件流：

  • 分为字符流和二进制流

使用：

mode meaning
r read
w write
a create or continue
r+ read and write
w+ read and write
a+ create or continue, read and wirte

note: binary flow needs a “b” after the mode

eg. “rb”, “a+b”
